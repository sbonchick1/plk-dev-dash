Claude handles everything — bugs, design, data, documents
🐛 Bug Fix
"The chart tooltips glitch when hovering — they flicker and show both datasets at once."
Found 2 root causes in one pass: tooltip firing on both datasets + afterDatasetsDraw running per-dataset instead of per-render. Fixed across all 16 charts instantly.
🎨 Layout Redesign
"The legend is overlapping with the data labels on the EBITDA chart. Let's move it so it doesn't overlap."
Moved the legend to position: 'bottom' in the shared bar() helper + added top padding. Change applied automatically to all grouped bar charts.
📰 Document Creation
"Here is my Canva newsletter — Canva doesn't handle tables well. Can you recreate this as a Word doc?"
Read the uploaded newsletter images, rebuilt the full 2-page April 2026 PLK Development Newsletter as a formatted .docx — red headers, opening tables, bullet sections, all preserved.
♻️ Code Refactor
"The CSS is duplicated across 5 HTML files. Can we extract it into one shared stylesheet?"
Created shared.css — 133 lines covering reset, KPI banner, tables, modals, risk mix, export button, and responsive breakpoints. Each dashboard file trimmed from ~200 lines to ~38 lines.
🏗️ Architecture
"The text is in text boxes, not inside the actual Excel table cells. Fix it for the Upside, Early Stages, and Prospect tables."
Diagnosed column width overflow (cols summed to 4.93" vs 4.88" available). Recalculated all three table layouts with autofit logic — text now renders properly inside every cell.
📊 Data Migration
"I want to migrate my dashboard from Render to Vercel. My code is on GitHub — walk me through it."
Audited the full repo, rewrote vercel.json with correct rewrites for all 6 routes, identified the public/index.html naming conflict causing 404s, and got it live in one session.
